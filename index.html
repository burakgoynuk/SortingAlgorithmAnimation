Selection Sort

Selection sort finds the minimum value, swaps it with the value in the first position, and repeats these steps for the remainder of the list. It does no more than n swaps, and thus is useful where swapping is very expensive. It has all best, worst, and average complexity with n*n.

Insertion Sort

Insertion sort works by taking elements from the list one by one and inserting them in their correct position into a new sorted list. In arrays, the new list and the remaining elements can share the array's space, but insertion is expensive.

Quick Sort

Quicksort is a divide and conquer algorithm which divides list according to selected pivot. Elements smaller than the pivot are moved before it and all greater elements are moved after it, then both are sorted recursively with the same process.

Merge Sort

Merge sort divides the unsorted list into n sublists, each containing 1 element (a list of 1 element is considered sorted). Repeatedly merge sublists to produce new sorted sublists until there is only 1 sublist remaining. This will be the sorted list.

Count Sort

Count sort works for only upper bound of the target data is defined. It creates a new array with length this bound and maps all elements to this array. Finally, it transfer new array to old array with exact location. It has O(n) time complexity.
